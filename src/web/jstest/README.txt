## To run tests with Karma:

- navigate to the test root directory (e.g. `shelob`) which contains a karma configuration file and a package.json file
- run `npm install` to install the dependencies listed in package.json
- run `karma start <karma.conf>` to run the tests as described in the karma configuration file

## For more information

See the angularjs tutorial: http://docs.angularjs.org/tutorial/
